Было предложена следующая задача

Тестовое задание
1) Перейти на страницу https://www.mos.ru/
2) Проверить наличие шапки подвала.
3) Вытащить все ссылки со страницы и проверить их на 200 (280 шт.)
4) Открыть каждую ссылку и проверить адресную строку браузера, что открывается нужная ссылка

После уточнения задачи, в следующих пунктах договорились сделать следующую проверку:
2) проверить на displayed()
3) ссылкой считается //a[@href] (этот локатор находит на странице https://www.mos.ru/ - 237 элементов)

Структура проекта
 Page Object страницы расположен в файле src\pages\main_page.py
 Тесты для выполнения задания расположены в файле tests\test_site.py

Файл tests\test_site.py содержит класс с 4 тестовыми методами

Запуск тестов:
Перейти в корневую папку с проектом
Ввести в консоли команду:
python -m pytest --alluredir=%absolute_path_to_directory_for_allure_results_report%

Для просмотра отчета о пройденных тестах
Ввести в консоли команду:
  allure serve %absolute_path_to_directory_for_allure_results_report%
